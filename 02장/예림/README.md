- 개발자 문서에 일반적으로 나타나는 콘텐츠 유형을 소개한다.
### 1. 코드 주석
- 과거의 내린 결정의 맥락을 남겨두면 훗날 코드를 볼 개발자들이 시간을 많이 절약할 수 있다
- 이에 반대하는 입장도 있지만 코드 주석은 코드가 무슨 일을 하는지에 대한 혼란과 모호성을 줄여주고 정보를 제공한다
